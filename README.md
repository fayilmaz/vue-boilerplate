# vue-boilerplate

## Available Scripts

In the project directory, you can run:

### Installs dependencies:

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your unit tests

```
npm run test:unit
```

### Dependencies:

Vue  
Vuex  
Vue Router  
Tailwind CSS  
Daisy UI  
Vee Vaildate  
Yup
